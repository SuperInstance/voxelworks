# VoxelWorks — Agent Instructions

## Overview
VoxelWorks is a game about making games. Fork-first game dev for kids. 
Build worlds with Scratch-like blocks, prompt-to-asset pipeline, and a Phaser.js game engine.

## Pages
- **hub-room/** — Main hub with Buddy chatbot
- **block-editor/** — Scratch-like visual programming blocks
- **asset-lab/** — Prompt-to-asset gallery  
- **ship-deck/** — Git-inspired deployment visualization  
- **game-template/** — Phaser.js platformer game engine (3 levels)

## Deployment
- Cloudflare Pages: Each page deployed as its own project
- GitHub: https://github.com/SuperInstance/voxelworks
